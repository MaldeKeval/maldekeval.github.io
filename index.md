## Projects

- **5G Testbed** — experimental 5G networking and RAN work at the lab
- **CCRAN** — converged cloud RAN: cloud radio access, split processing, and related system studies

## Core strengths

- **RAN software engineering:** MAC feature development, scheduler tuning, and protocol-level implementation
- **Performance optimization:** CPU/thread profiling, cache and power analysis, and deployment-specific tuning
- **Systems integration:** UE L2/L3 and L1 integration, shared-memory/DPDK-based interfaces, and end-to-end call-flow analysis
- **Automation and tooling:** Python and shell automation for repetitive operations, data exploration, and validation workflows
